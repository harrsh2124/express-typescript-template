# Express TypeScript Boilerplate

Express Boilerplate to use for all typescript/node project as api.


## Installation

Install on development

```bash
  yarn 

  yarn dev
```

Install on production

```bash
  yarn 

  yarn build

  yarn start
```


## Tech Stack

**Client:** hbs

**Server:** Node, Express, Typescript, MySql/PostgreSql
